2017-02-20
技术文档

数据单向流动
view -> action -> store -> view

store 通过register函数向dispatcher注册事件，定义事件名和事件处理逻辑。

AppDispatcher.register(function (action) {
  switch (action.actionType) {
    case AppsEvents.REQUEST_APPS:
      storeData.apps = flatList(action.data.body);
      AppsStore.emit(AppsEvents.CHANGE);
      break;
      ...

action 获取数据，通过 dispatcher 分发对应的数据到 store 上。

this.request({
  method: "POST",
  data: {
    force: force
  },
  url: url
})
  .success(function (app) {
    AppDispatcher.dispatch({
      actionType: AppsEvents.RESTART_APP,
      data: app,
      appId: appId
    });
  })

view 通过store的on函数注册回调函数，获取数据，渲染新的视图。

componentWillMount: function () {
  AppsStore.on(AppsEvents.CHANGE, this.onAppsChange);
},
onAppsChange: function () {
  this.setState({
    apps: AppsStore.apps,
    fetchState: States.STATE_SUCCESS
  });
},
